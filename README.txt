Requisitos previos del proyecto:

Python 3:

link de descarga: “https://www.python.org/downloads/”

Es recomendable habilitar la opción "Add Python to PATH" durante la instalación para evitar configurarlo manualmente

Flask:
(para descargarla es necesario tener Python previamente instalado)

Pegar este comando en CMD para instalar flask: “pip install flask”

Ejecución del proyecto:
Para ejecutar el proyecto es necesario ejecutar desde la terminal el archivo “backend/app.py”
Pegar en terminal: “python app.py”
El servidor comenzara a correr en un una ip que se indicara en consola, siendo http://127.0.0.1:5000 la ip por defecto, al entrar a esta ip desde el navegador se mostrara la interfaz de la aplicación
